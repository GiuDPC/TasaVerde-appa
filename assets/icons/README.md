# Carpeta de Iconos SVG

## Iconos necesarios:

| Archivo          | Uso                                 |
| ---------------- | ----------------------------------- |
| `bcv.svg`        | Tasa BCV en dashboard               |
| `binance.svg`    | Tasa Binance en dashboard           |
| `euro.svg`       | Tasa Euro en dashboard              |
| `lightbulb.svg`  | Indicador "Mejor opción"            |
| `calculator.svg` | Icono tab calculadora               |
| `chart.svg`      | Icono tab tasas                     |
| `arrow-down.svg` | Indicador "Desliza para actualizar" |

## Instrucciones:

1. Descarga o crea tus iconos SVG
2. Colócalos en esta carpeta
3. Avísame cuando estén listos para integrarlos en la app
